# ShoeSense AI

Website untuk menganalisis tingkat kebersihan sepatu dan bahan sepatu berbasis AI.

## Fitur

- Upload gambar sepatu
- Kamera langsung
- Analisis kebersihan
- Deteksi bahan sepatu
- Riwayat analisis

## Teknologi

- Frontend: Vue 3 + Vite
- Backend: Flask + TensorFlow/Keras
- Model: CNN Multilabel
